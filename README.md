# Cloud Desktop System

A Linux desktop in the cloud: sign in via the browser (Apache Guacamole) and use a full XFCE desktop. Each user gets their own Docker-based desktop.

## Quick start (on Google Cloud VM)

1. **Create a VM** on Google Cloud: Ubuntu 22.04, e2-standard-2, 80 GB disk, allow HTTP/HTTPS.
2. **Copy this project** to the VM (e.g. `git clone` or `scp`).
3. **SSH** into the VM and run:
   ```bash
   cd /path/to/terabits-1.2
   sudo bash scripts/setup-vm.sh
   ```
4. Open **http://YOUR_VM_IP:8080/guacamole/** and log in as `guacadmin` / `guacadmin`.
5. Create a Guacamole user, then give them a desktop:
   ```bash
   pip install -r scripts/requirements.txt   # or pip3
   python3 scripts/manage_desktops.py create USERNAME
   ```

Full steps: see [docs/SETUP.md](docs/SETUP.md).

## Project layout

- `docker-compose.yml` — Guacamole (guacd + web + PostgreSQL)
- `desktop-image/` — Dockerfile and scripts for the XFCE+VNC desktop
- `scripts/setup-vm.sh` — One-time VM setup (Docker, images, Guacamole)
- `scripts/manage_desktops.py` — Create/delete desktop containers and register in Guacamole
- `nginx/` — Optional reverse proxy config for HTTPS
- `init/` — Guacamole DB schema (generated by setup-vm.sh)
- `docs/SETUP.md` — Detailed setup and troubleshooting
